List of applications and tools on my macOS

## Setup

### Xcode

```sh
xcode-select --install
```

### [Homebrew](https://brew.sh/) The Missing Package Manager

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
